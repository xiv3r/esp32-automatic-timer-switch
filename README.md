# Requirements
- ESP32 30P/38P Pins
- 8-Channel Relay
- 10 Dupont Wire

# Arduino Libraries
- ArduinoJson
- NTPClient

# Installation
- Download the firmware and flash using `0x0` offset

# Pinouts
```
RELAY     ESP32 38P
VCC _____ 5VIN 
IN1 _____ D32 GPIO
IN2 _____ D33 GPIO
IN3 _____ D25 GPIO
IN4 _____ D26 GPIO
IN5 _____ D27 GPIO
IN6 _____ D14 GPIO
IN7 _____ D16 GPIO
IN8 _____ D17 GPIO
GND _____ GND
```
